## 5단계: `Places` 시트 설정

### 헤더 (1행)

```
name	region	district	address	lat	lng	category1	category2	tag1	tag2	tag3	brand1	brand2	brand3	hours_text	closed_days	description	instagram_url	naver_map_url	status
```

### 데이터 유효성 검사

- B열(region): 항목 목록 `서울, 경기, 인천, 강원, 충청, 경상, 전라, 제주, 대구`
- G열(category1), H열(category2): 범위 `Categories!B2:B10`
- I열(tag1), J열(tag2), K열(tag3): 범위 `Tags!B2:B40`
- L열(brand1), M열(brand2), N열(brand3): 범위 `Brands!B2:B30`
- T열(status): 항목 목록 `OPEN, CLOSED, RELOCATED`
