# PRD: B-02 Place 도메인
- 작성일: 2026-04-05
- 완료일: -
- 상태: Draft

## 배경 & 목적

서비스의 핵심 도메인. 뜨개 관련 장소(뜨개샵, 공방, 뜨개카페 등)를 조회하는 API를 제공한다.
프론트엔드가 장소 목록 화면과 상세 화면을 구성하는 데 필요한 데이터를 반환한다.

## 요구사항

### 기능 요구사항

- [ ] 장소 목록 조회 — 전체 목록 반환, 지역/카테고리/태그 필터링 지원
- [ ] 장소 상세 조회 — 단일 장소의 전체 정보 반환 (카테고리, 태그, 브랜드 포함)
- [ ] 상태가 OPEN / CLOSED / RELOCATED 인 장소 모두 반환 (상태값 포함)

### 비기능 요구사항

- 인증 불필요 (공개 API)
- 목록 응답은 상세보다 가벼운 필드 구성

## 가정 & 미확정 사항

- 가정: 페이지네이션은 이번 범위 밖 (데이터 적음)
- 가정: 검색(키워드)은 이번 범위 밖
- 가정: 정렬 기준은 id ASC (등록 순)
- 미확정: 목록에서 브랜드 노출 여부 → 일단 미포함 (상세에서만)

## 범위 외

- 장소 등록/수정/삭제 (관리자 기능)
- 페이지네이션
- 키워드 검색
- 즐겨찾기
