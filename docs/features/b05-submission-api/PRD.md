# PRD: B-05 제보 API
- 작성일: 2026-04-05
- 완료일: 2026-04-05
- 상태: Done

## 배경 & 목적

사용자가 앱에서 직접 장소/이벤트를 제보할 수 있어야 한다.
제보는 PENDING 상태로 저장되고, 관리자가 확인 후 승인/거절한다.

## 요구사항

| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| R1 | POST /api/submissions/places — 장소 제보 등록 | Must |
| R2 | POST /api/submissions/events — 이벤트 제보 등록 | Must |
| R3 | 성공 시 201 Created + 생성된 id 반환 | Must |

## 범위 제외

- 관리자 승인/거절 (B-06)
- 인증 (B-07) — 현재는 인증 없이 누구나 제보 가능
