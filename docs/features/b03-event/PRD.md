## 배경 & 목적

뜨개 관련 이벤트(팝업, 세일, 테스터 모집)를 사용자가 조회할 수 있어야 한다.
장소(Place)와 독립적으로 존재할 수 있으며, 기존 장소와 연결되거나 임시 장소 텍스트로 표현한다.

## 요구사항

| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| R1 | 이벤트 목록 조회 (이벤트 타입 / active 필터) | Must |
| R2 | 이벤트 상세 조회 | Must |
| R3 | 존재하지 않는 id 조회 시 404 반환 | Must |

## 범위 제외

- 이벤트 생성/수정/삭제 (관리자 기능, B-06 이후)
- 인증/인가 (B-07)
