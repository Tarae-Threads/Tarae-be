# PRD: 관리자 세션 로그인
- 작성일: 2026-04-19
- 완료일: 2026-04-19
- 상태: Done

## 배경 & 목적

현재 `/admin/**` 경로가 인증 없이 누구나 접근 가능한 상태.
관리자 페이지(Thymeleaf SSR)에 세션 기반 로그인을 추가해 무단 접근을 차단한다.

## 요구사항

### 기능 요구사항
- [ ] 커스텀 로그인 페이지(`/admin/login`) 제공
- [ ] 아이디/비밀번호로 로그인, 성공 시 대시보드로 이동
- [ ] 로그인 실패 시 오류 메시지 표시
- [ ] 로그아웃 기능 (`/admin/logout`)
- [ ] `/admin/**` 미인증 접근 시 로그인 페이지로 리다이렉트
- [ ] 세션 유효시간 3시간 (비활성 기준)

### 비기능 요구사항
- 관리자 계정(ID/PW)은 환경변수로 관리 (`ADMIN_USERNAME`, `ADMIN_PASSWORD`)
- 비밀번호는 BCrypt 해시로 비교
- Remember-me 없음
- 동시 세션 제한 없음
- `/api/**`는 현행 인증 정책(permitAll) 유지

## 가정 & 미확정 사항
- 가정: 관리자는 1명, DB 계정 테이블 불필요
- 가정: 프로덕션 환경에서 HTTPS 사용 (Secure 쿠키)
- 가정: CSRF는 관리자 폼 경로에만 활성화, API 경로는 비활성 유지

## 범위 외
- 관리자 계정 추가/변경 UI
- 로그인 이력/감사 로그
- 2FA
