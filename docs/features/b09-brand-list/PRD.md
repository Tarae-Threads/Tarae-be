- 완료일: 2026-04-11

## 배경 & 목적
프론트엔드에서 장소 등록/검색 시 브랜드를 선택할 수 있어야 한다.
브랜드를 타입별로 그룹핑해서 내려주면 탭/섹션 UI 구성이 용이하다.

## 요구사항
### 기능 요구사항
- [ ] 전체 브랜드를 타입별로 그룹핑하여 반환한다
- [ ] 타입 순서: YARN → NEEDLE → NOTIONS → PATTERNBOOK
- [ ] 각 브랜드 항목은 id, name을 포함한다

### 비기능 요구사항
- 인증 불필요 (공개 API)

## 가정 & 미확정 사항
- 가정: 브랜드 수가 많지 않아 페이징 불필요
- 미확정: 타입 순서 고정 여부 (→ 일단 enum 선언 순서 따름)

## 범위 외
- 브랜드 생성/수정/삭제
- 브랜드 검색
